Transit Terminal Departures Board

A real-time C terminal application — Standard C99 • Linux / Seneca Matrix

C99 GCC Linux POSIX

Compile & Run

# Build with make
make

# Or compile directly
gcc -Wall -std=c99 -o departures departures.c

# Run (uses routes.txt by default)
./departures

# Run with a custom routes file
./departures my_routes.txt

Routes File Format

# ROUTE_NUMBER  STOP_NAME  FREQUENCY_MINS  OFFSET_SECS
501  Queen    10    0
504  King      8   90
510  Spadina  12   45

Features

FeatureDescription
Live countdownUpdates every second — "Arriving in 4 min", "Due in 52 sec", "NOW DEPARTING"
Colour-coded rowsCyan (far), Green (<5 min), Yellow (<1 min), Red (departing)
Sorted boardAll routes merged and sorted by next arrival time
60-min windowShows the next 60 minutes of departures, 15 rows at a time
Clean exitCtrl+C restores cursor and prints a goodbye message
